Beaufort Cracker

Break Beaufort cipher using a dictionary attack. Similar to Vigenère but uses subtraction rather than addition.
About Beaufort Encrypts as C = (K − P) mod 26 instead of Vigenère's (P + K) mod 26. It is self-reciprocal — encrypting twice returns the original plaintext. This tool tries every word in the dictionary as a potential key.
