Anagram Solver

Solve pattern-based anagram puzzles in pre- or post-anagram mode with dictionary verification.
Pre-Anagram The pattern groups letters by position. Same letter in the pattern = letters belong to the same output group.
Example: text sodnhtfdeeci pattern ccaabbacbcca
Post-Anagram Each unique pattern letter represents one word. Same pattern letter = same word.
Example: text tobolikl pattern aaaabbbb gives two 4-letter words.
